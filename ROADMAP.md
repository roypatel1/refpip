# RefPip Roadmap

RefPip (Reference Picture-in-Picture) is evolving from a Chrome Extension to a cross-platform productivity suite. Our goal is to bring the "Picture-in-Picture" experience to any content, anywhere.

## Phase 1: Chrome Extension Excellence (Current)
- [x] **v1.0: MVP** - Pin AI chat blocks in draggable portals.
- [x] **v1.1: Selection Mode** - Pin any text selection on any website.
- [x] **v1.2: Screen Mirroring** - Snapshot and pin the current viewport.
- [x] **v1.3: Dark Mode Support** - Themed portals for dark-mode websites.
- [ ] **v1.4: Persistence** - Save portals across browser restarts.
- [ ] **v1.5: Cloud Sync** - Sync pinned content across different machines.

## Phase 2: Browser Expansion (Q3 2026)
- [ ] **Firefox & Safari Support** - Port the extension to all major browsers.
- [ ] **Edge Optimized** - Deep integration with Edge's sidebar.
- [ ] **Mobile Companion** - A mobile app to view your pinned "RefPips" on the go.

## Phase 3: Desktop Native (Q4 2026)
- [ ] **RefPip Desktop (Windows/macOS/Linux)** - A standalone app that allows pinning content *outside* the browser.
- [ ] **System-Wide Selection** - Use OCR to pin text from non-selectable areas (PDFs, Images, Videos).
- [ ] **Always-on-Top Overlays** - True system-level PiP that stays above all windows, not just browser tabs.

## Phase 4: AI & Automation (2027)
- [ ] **Smart Summarization** - Automatically summarize pinned content using Gemini.
- [ ] **Contextual Linking** - Automatically find related documents or chats based on what you've pinned.
- [ ] **Voice Interaction** - Ask questions about your pinned content.
