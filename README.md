# RefPip - Reference Picture-in-Picture

**RefPip** is a Chrome Extension that solves "Long-Thread Navigation" in AI Chat interfaces (Gemini, AI Studio, ChatGPT). It allows users to "pin" specific chat blocks into draggable, resizable, and scrollable floating "portals" within the same browser tab for constant reference.

## 🚀 Features
- **Pin Chat Blocks**: Instantly clone and pin AI responses or user queries.
- **Floating Portals**: Draggable, resizable, and scrollable windows that stay on top.
- **Screen Mirroring**: Capture and pin your current viewport for manual reference.
- **Element Selection**: Precisely select any HTML element to pin.
- **Dark Mode Support**: Portals adapt to your preferred theme.
- **Local Processing**: Your data never leaves your machine.

## 🛠 Installation (Developer Mode)
1.  **Download** the latest release zip from [refpip.io](https://refpip.io).
2.  **Unzip** the files to a local folder.
3.  Open Chrome and navigate to `chrome://extensions`.
4.  Enable **Developer Mode** (top right toggle).
5.  Click **Load unpacked** and select the folder where you unzipped RefPip.
